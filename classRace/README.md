### 使用到的技术

- Webpack + ES6 + Vue1.x + Vue-Router + Vuex
- ThinkPHP3.2 + MySql
- 基于[微信接口](https://mp.weixin.qq.com/wiki?t=resource/res_main&id=mp1421135319)的后端开发
